# CSharp

## 环境搭建

我使用 VSCode 来搭建开发环境。

在 VSCode 里下载 [C# Dev Kit](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csdevkit) 拓展，它会自动打开一个引导界面，帮助你获取工具链并配置插件，跟着提示一步步完成就可以了

详细过程可参考 [VSCode C# 官方文档](https://code.visualstudio.com/docs/csharp/get-started)

> [!Note]- 解释器
> C# 有个解释器 `scriptcs`。不推荐用它来编写实际项目，但可以拿来学习和探索。
>
> 在 Windows 上可通过 `scoop install scriptcs` 安装
