# Eza

## 简介

eza 是个文件目录查看工具，类似 ls，但更现代

## 对比

eza 相比 ls，我个人认为有如下的优点

- 更好的默认设置。比如 eza 默认输出就是有颜色的，而 ls 还得自己加参数
- 更丰富的功能
- 单一二进制文件，分发更简单

还有个叫 lsd 的工具，也自称是 ls 的现代替代品。但经过我的简单比较后发现 lsd 比 eza 慢一点，就一直使用后者了

## 安装

Windows 上可以通过 Scoop 安装

```sh
scoop install eza
```

## 使用

只列出我常用的命令

```sh
# 直接使用
eza
# 包括隐藏文件
eza -a
# 以列表形式呈现
eza -l
# 给列表加个表头
eza -l -h
# 显示图标
eza --icons
# 递归查看
eza -R
# 以树形式呈现（会自动开启递归）
eza -T
```
