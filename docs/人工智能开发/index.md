# 人工智能开发

**机器学习**掀起了人工智能的新浪潮，以至于现在基本变成了 *AI* 的同义词。所以我不打算介绍传统的人工智能技术，后面的一切基本都和机器学习有关。

机器学习和数据科学/科学计算等领域联系紧密，本文也会包含这些主题 ~~所以本部分的完整名称应该是人工智能/数据科学/科学计算开发~~。

由于 Python 几乎已经成为了人工智能领域的标准语言，所以后面的框架大多和 Python 有关。Python 还有一些通用的框架/库/工具，由于不是专属于 AI 的，所以没有列举在这里，可以在 [Python](../编程语言/Python.md) 章节查看

## 框架

- [PyTorch](../库和框架/PyTorch.md) 官方说法是 `一个用于深度学习的、优化过的张量库`，不过社区习惯称其为机器学习框架
- TensorFlow Google 开发的机器学习框架，不过 PyTorch 在学术界更受欢迎
- [YOLO](../库和框架/YOLO.md) 基于 *PyTorch* 的深度学习框架，适合图像识别应用，能让你迅速完成从数据准备、模型训练、模型验证到模型导出的整个过程，并且提供了 CLI
- Kedro 数据科学和机器学习的管道框架，用于将项目从 Jupyter 加孤立脚本的混乱状态过渡到可实际投产的状态

## 库

- scikit-learn 机器学习库，实现了非常多的基本算法，使用了 numpy、scipy 和 matplotlib 来实现
- sympy 符号计算库
- numpy 数值计算库
- scipy 科学计算库
- matplotlib 绘图库
- pandas 数据分析库，用于分析表格型数据
- networkx 数据分析库，用于分析图状数据
- control 一个Python 的控制系统库。虽然现在机器人等领域更倾向于使用 *强化学习* 等新技术，但传统的控制技术由于可解释性更强，也还是有用的。

## 工具

- DVC 数据版本控制，基于 Git 但巧妙地处理了大文件，用于开发可复现的机器学习项目
